Following criteria should be followed while setting Password

  • Password should contain atleast 1 Capital letter (A-Z)
  • Password should contain atleast 1 Small letter (a-z)
  • Password should contain atleast 1 Special charecter (@,#,$,% etc)
  • Password should contain atleast 1 number (0-9)
  • Password should be greater than 8 charecters and less than or equal to 16 charecters
